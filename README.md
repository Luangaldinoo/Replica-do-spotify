---

## 📋 Sobre o Projeto

Este projeto consiste na **réplica da interface web do Spotify**, desenvolvida como parte dos estudos em desenvolvimento front-end.

O foco principal está na **estruturação semântica com HTML5** e na **estilização moderna com CSS3**, aplicando conceitos como Flexbox, organização de layout, posicionamento de elementos e efeitos visuais interativos.

O objetivo é demonstrar domínio em:

- Estruturação de páginas web  
- Organização de layout  
- Criação de componentes visuais  
- Responsividade básica  
- Boas práticas de front-end  

---

## ✨ Funcionalidades

- **Sidebar Navegável**: Menu lateral com navegação para Home, Search e Sua Biblioteca  
- **Seção Sua Biblioteca**: Área para criação de playlists e exploração de podcasts  
- **Navegação Superior**: Botões de voltar e avançar, além de opções de inscrição e login  
- **Músicas em Alta**: Cards com capa, título, artista e botão de play interativo ao passar o mouse  
- **Artistas Populares**: Cards circulares destacando artistas  
- **Seção Rádio**: Layout adicional com exibição de conteúdos em formato de cards  
- **Rodapé Completo**: Múltiplas colunas de links institucionais e informativos  
- **Links para Redes Sociais**: Ícones de Instagram, X (Twitter) e Facebook  
- **Estrutura Responsiva (Inicial)**: Base preparada para adaptação a diferentes tamanhos de tela  

---

## 🗂️ Estrutura do Projeto

### Arquivos Principais

- **index.html**: Estrutura principal da aplicação  
- **style.css**: Arquivo responsável por toda estilização da interface  
- **assets/**: Pasta destinada a imagens e ícones utilizados no layout

## 🚀 Como Usar

### Pré-requisitos

- Navegador atualizado (Google Chrome, Edge, Firefox, etc.)  

### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/Luangaldinoo/Replica-do-spotify.git
   ```

2. **Acesse a pasta do projeto**
   ```bash
   cd Replica-do-spotify
   ```

3. **Abra o arquivo no navegador**
   ```bash
   index.html
   ```

Ou simplesmente clique duas vezes no arquivo `index.html`.
