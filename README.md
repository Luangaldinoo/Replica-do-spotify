🎧 Spotify Clone (Réplica de Interface Web)
Um projeto desenvolvido como parte dos estudos de desenvolvimento web, recriando a interface de usuário do famoso serviço de streaming de música, Spotify.

Este clone foca na estrutura e no design visual da plataforma, utilizando HTML e CSS puros para demonstrar habilidades de front-end.


✨ Recursos Principais
Sidebar Navegável: Menu lateral com ícones e links para "Home", "Search" e "Sua Biblioteca".
Seção "Sua Biblioteca": Inclui áreas para "Criar Playlist" e "Explore Podcasts" com botões interativos.
Navegação Principal: Botões de "Voltar" e "Avançar" no topo, além de opções de "Inscrever-se" e "Entrar".
Seções de Conteúdo Dinâmico:
Músicas em Alta: Exibição de cards de músicas com capa, título e artista, e um botão de play que aparece ao passar o mouse.
Artistas Populares: Cards redondos para artistas com nome e tipo (Artista).
Rádio: Uma seção adicional com a mesma estrutura de cards de músicas.
Rodapé Completo:
Múltiplas colunas de links ("Empresa", "Comunidades", "Recursos", "Planos Premium").
Links para informações importantes (Legal, Segurança, Política de Privacidade, Cookies, etc.).
Opção para selecionar o idioma ("Português do Brasil").
Links para Redes Sociais: Ícones de Instagram, X (Twitter) e Facebook.
Design Responsivo (Básico): Estrutura inicial para adaptação a diferentes tamanhos de tela (embora focado em desktop neste momento).

🛠️ Tecnologias Utilizadas
HTML5: Estruturação do conteúdo da página.
CSS3: Estilização e layout da interface, incluindo flexbox para organização.


Abra no Navegador:
Basta abrir o arquivo index.html diretamente no seu navegador de preferência.
